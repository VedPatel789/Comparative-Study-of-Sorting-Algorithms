Sorting Algorithm Benchmark (Python)

A Python project that implements and compares the performance of multiple sorting algorithms using execution time analysis and visual graphs.

Features
Implements 8 sorting algorithms:
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
Heap Sort
Shell Sort
Counting Sort
Benchmarks execution time using time.perf_counter()
Tests algorithms on different input sizes
Visual comparison using Matplotlib graphs
